## Feature Breakdown

### 📊 Country Admin Statistics
- View total hours spent by users from each country
- See number of sessions per country
- Count unique users per country
- Color-coded visualization (red > 100h, orange > 50h, green < 50h)

### 📝 Admin Time Logs
- Track individual user admin sessions
- See login/logout times
- Auto-calculate session duration
- Filter by user, country, or date

### ⚙️ Automation
- **Middleware**: Automatically logs when users enter/exit admin
- **Management Command**: Periodically recalculate country stats
- **Aggregation**: Uses Django ORM to sum up hours by country

## How It Works

```
User visits /admin/ 
        ↓
AdminTrackingMiddleware.process_request() 
        ↓
Creates new AdminTimeLog(user, login_time)
        ↓
User browses admin interface
        ↓
User visits non-admin page
        ↓
AdminTrackingMiddleware.process_response()
        ↓
Updates AdminTimeLog(logout_time, duration_seconds)
        ↓
Run: python manage.py recalculate_country_stats
        ↓
CountryAdminStats updated with aggregated data
```

## Access Points

### Django Admin Views:
1. **AdminTimeLog** - View individual sessions
   - Path: `/admin/users/admintimelog/`
   - Shows: User, Login, Logout, Duration, Status
   - Filters: By user, country, date

2. **CountryAdminStats** - View country statistics
   - Path: `/admin/users/countryadminstats/`  
   - Shows: Country, Hours, Sessions, User Count
   - Summary: Total countries, users, hours, average

### Management Commands:
```bash
# View all sessions
python3 manage.py recalculate_country_stats

# Clear and recalculate all
python3 manage.py recalculate_country_stats --clear
```
